60-second quickstart

Save the key, then say hi to a model:

# 1. Save the key (don't commit it, don't paste it in chat)
export AI_GATEWAY_API_KEY="<the-key-you-were-given>"

# 2. Say hi
curl -X POST "https://ai-gateway-eastus2.azure-api.net/openai/v1/responses" \
  -H "api-key: $AI_GATEWAY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-5.4-nano","input":"Say hi to the hackathon."}'

JSON back = you’re in. 401 = wrong key. 429 = slow down, speedy.

House rules

  1. One key, one human. Don’t share keys with your team — ask for more, they’re free.
